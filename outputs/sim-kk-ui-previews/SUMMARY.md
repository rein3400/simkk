## How to regenerate

```bash
cd outputs/sim-kk-ui-previews
node screenshot.cjs
```

Requires `playwright` to be installed under `apps/web/node_modules/playwright`.

## Notes

- Screens are static HTML previews — they are not connected to the API.
- Tailwind is pre-built to `assets/tailwind-built.css`; do not edit the source
  HTML/Tailwind config without a corresponding design pass.
- Login is the only screen without a PRD section number — it is the entry
  surface, not a feature deliverable.
